# 漢字オセロ

## 概要
テレビ番組風クイズの初作品。TBS系列で放送されている『東大王』の人気企画、難問オセロを模したオセロゲーム。<br>
※srcディレクトリにビルド前のコード、distディレクトリにビルド後のコードが入っています。

## 開発環境・使用言語
### 開発環境
- macOS
- VS Code
- webpack
### 使用言語
- HTML
- SCSS
- JavaScript

## 今回の学び
1. WebサイトのUI部品に使用するJavaScriptでは身につかない、長い処理の流れを意識した開発を経験することができた。
2. オセロというルールが決まっているゲームを自力で再現することができ、少し自信がついた。<br>
今後はチェスのようにコマ毎に役割が異なる複雑なゲームの開発にも挑戦したい。

## 今後の改善点
1. 石を返す処理にアニメーションをつけるなど、デザイン面を改善し、よりゲームらしく修正していきたい。
2. 現在は１台の端末を用いたオフラインでの対戦機能のみだが、オンラインでの対戦機能を実装することが目標。
